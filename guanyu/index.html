        # 伊蕾娜bot是什么

        伊蕾娜bot是基于nonebot框架，以HoshinoBot与Yobot为主体进行修改的，综合一系列HoshinoBot插件实现的多功能性质的QQ群bot。

        # 我该如何获得这样的机器人

        HoshinoBot与Yobot均为GitHub开源项目
        您可根据GitHub上作者提供的部署教程自己在本地或是服务器部署一个属于你的BOT。
        以下为源码地址：
        HoshinoBot：https://github.com/Ice-Cirno/HoshinoBot (opens new window)
        Yobot：https://github.com/yuudi/yobot (opens new window)

        # 我不想/不会自己动手搭建，有什么其他获取渠道吗

        您可以通过页面右上角的【捐赠】按钮获取伊蕾娜bot。
        伊蕾娜bot是本人自行搭建在云服务器上的bot，由于服务器与bot维护需要成本，因此需要您的赞助体验，感谢您的赞助，您的每一分赞助都将用于伊蕾娜bot的维护。

        # 我能用伊蕾娜bot做什么

        您可以通过伊蕾娜bot进行包括但不限于：群管理、公主连结会战报刀记录与合刀计算、原神自动签到、原神数据查询、马娘数据查询、娱乐等。
        请在日常调教bot时注意分寸，以免bot刷屏导致tx风控！
